环境准备

  • JDK 1.8+

  • Maven 3.2.x

  • Git

代码拉取

  > git clone https://github.com/yu199195/hmily.git

  > cd hmily

  > mvn -DskipTests clean install -U

执行demo 模块的sql语句。

[sql语句] (https://github.com/yu199195/hmily/blob/master/hmily-demo/sql/hmily-demo.sql)

使用你的工具 idea 或者eclipse 打开项目。
