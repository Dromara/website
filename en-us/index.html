        Introduce

        Soul is an asynchronous, high-performance, cross-language, responsive API gateway that provides uniform HTTP access, seamless support for dubbo,Spring cloud, and a wealth of plug-ins (current limiting, fuse, proxy forwarding)

        Features

        • Cross-language

          Support for all languages

        • More plugins

          Authentication, current limit, fuse, firewall and other plugins

        • Dynamic Config

          All of the configuration rules can be adjusted at will, taking effect dynamically, without restarting

        • Plugins hot-plug, easy to expand

          Plugins hot - plug, easy to expand

        • Support More Rpc and REST

          Dubbo，Spring cloud，HTTP REST

        • High availability, high concurrency

          Support cluster deployment
